git init : erstellt ein leeres repository
git add : fügt datein in die stagin area
git commit: fügt datein von der stagin area ins repository
git checkout: wechselt die position des headers
git branch: erstellt einen branch
git merge: fügt zwei branches zusammen
git log: zeigt den git verlauf an
git status: zeigt den status den branches an
git rebase: commits positon verändern
git reset: macht den header zurück und entfert die ausgewählten commits
git revert: erstellt einen neuen commit, der zurückgesetzt ist
git clone: kopiert das repository von einem remote-repository
git push: kopiert die änderungen des eigenen reps ins remote
git pull: kopiert die veränderungen aus dem remote rep zum eigenen und merged die
